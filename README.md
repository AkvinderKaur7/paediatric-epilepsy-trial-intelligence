# Paediatric Epilepsy Clinical Trial Intelligence Dashboard

## Project Overview
An independent data analytics project analysing 807 paediatric 
epilepsy clinical trials from ClinicalTrials.gov (1991–2026), 
built using SQL and Power BI.

## Key Findings
- Paediatric epilepsy research has grown 5x since 2000
- USA dominates with 44% of all global trials
- India contributes only 2.48% despite massive disease burden
- Phase 3 trials dominate — early innovation pipeline is weak
- 56.5% of trials completed — large evidence base exists

## Tools Used
- MySQL — data cleaning and analysis
- Power BI — interactive dashboard
- Excel — data preprocessing
- ClinicalTrials.gov — primary data source
